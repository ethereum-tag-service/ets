# ETS Smart Contracts

Welcome to the ETS Smart Contracts documentation. This section contains:

- **Hardhat Tasks** - a list of tasks that can be run from the command line for directly interacting with the ETS smart contracts.
- **@ethereum-tag-service/contracts package** - Docs for our comprehensive Typescript package providing utilities, constants and configurations for building applications on the Ethereum Tag Service (ETS) protocol.
- **Technical Referance** - Core Solidity smart contract documentation generated from NatSpec comments.
