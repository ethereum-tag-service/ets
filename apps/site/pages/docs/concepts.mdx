# Concepts

## Key participants

<table className="my-12">
  <tr>
     <td>Role</td>
     <td>Description</td>
  </tr>
  <tr>
    <td>**Publisher**</td>
    <td>Wallet address of whitelisted third-party service (dApp, smart contract, application) that implements ETS</td>
  </tr>
  <tr>
    <td>**Tagger**</td>
    <td>Wallet address credited with initiating a tagging event</td>
  </tr>
  <tr>
    <td>**Creator**</td>
    <td>Wallet address credited with creating a new tag</td>
  </tr>
  <tr>
    <td>**Sponsor**</td>
    <td>Wallet address that pays transaction fee for tagging or minting a tag</td>
  </tr>
  <tr>
    <td>**Auctioneer**</td>
    <td>Wallet address that supports initial sale of a tag</td>
  </tr>
  <tr>
    <td>**Owner**</td>
    <td>Wallet address that purchases a tag at auction or secondary market</td>
  </tr>
  <tr>
    <td>**Curator**</td>
    <td>Wallet address of user that verifies tags</td>
  </tr>
</table>
