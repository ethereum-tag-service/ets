# `ETS API` fetch CTAGs using Apollo Client

A simple example demonstrating an ETS API call using [Apollo Client](https://github.com/apollographql/apollo-client).

## run locally

```bash
pnpm install
pnpm run dev

# open Vite dev url in browser.
```
