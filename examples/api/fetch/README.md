# `ETS API` fetch CTAGs using Fetch

A simple example demonstrating an ETS API call using [Fetch](https://developer.mozilla.org/en-US/docs/Web/API/Window/fetch).

## run locally

```bash
pnpm install
pnpm run dev

# open Vite dev url in browser.
```
