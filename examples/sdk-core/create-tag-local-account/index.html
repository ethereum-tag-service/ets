ETS Examples: sdk-core create tagging record

After you click "Run Demo", you will be shown a dynamically created, single-use wallet address.

To continue the demo, fund that address with a tiny amount of Base Sepolia ETH.

After the wallet is funded, click Run Demo again, and this wallet will create tagging record with the following parameters: 

  const taggingRecord = await relayerClient.createTaggingRecord(
    ["#rainbow", "#unicorn"],
    "https://github.com/ethereum-tag-service/examples",
    "examples"
  );

You will be able to view this record in the ETS Base Sepolia Explorer.

Click 'Run Demo' to start
